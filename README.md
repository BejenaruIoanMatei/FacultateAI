# FacultateAI
Pentru Inteligenta Artificiala, anul 3 sem 1

Cerinta Proiect : 
Catology
Implement a system able to classify a cat as belonging to one breed. Main functionalities:
Translate and add new instances in the dataset available here: https://data.mendeley.com/datasets/ht5p5pg7b7/1 
Read a natural language description of a cat, extract relevant attributes and identify its breed.
Generate a natural language description of a cat breed using the trained classifiers. Generate a natural language comparison between two breeds.

Modeling task:  Exploratory analysis of the dataset.
NN/RL task: Train a classifier for a cat breed.
KR/NLP task: Generate a natural language text description of a breed using the trained classifiers.
